# The Impossible Puzzle (不可能谜题 / 弗赖登塔尔谜题)

本目录包含了认知逻辑中著名的“不可能谜题”的数学推导，以及使用 Python 集合过滤构建的算法解算器。

## 题目设定

有两个未知的整数 $x$ 和 $y$，满足 $1 < x < y$ 且 $x + y \le 100$。
- P 先生 (Mr. Product)：只知道这两个数的乘积 $P = x \cdot y$。
- S 先生 (Mr. Sum)：只知道这两个数的和 $S = x + y$。

两人都是绝对理性且极度聪明的数学家。接下来发生了如下四句对话：

1. P 说：“我不知道这两个数是多少。”
2. S 说：“我早就知道你不知道。”
3. P 说：“那我现在知道这两个数是多少了！”
4. S 说：“那我现在也知道了！”

目标：通过这四句话推导出唯一的 $x$ 和 $y$。

## 推演过程

我们定义初始的样本空间为所有满足 $1 < x < y$ 且 $x + y \le 100$ 的数对 $(x, y)$。

### 1. P 说“我不知道”
逻辑推导：如果 $P$ 唯一对应一组数对（即 $P$ 的因数分解只有一种满足条件的分法），P 立刻就能知道 $x$ 和 $y$。
空间坍缩：剔除所有乘积 $P$ 只有唯一分解方式的数对。

### 2. S 说“我早就知道你不知道”
逻辑推导：S 只知道和 $S$。这句话意味着：在所有满足加和为 $S$ 的数对中，没有任何一个数对的乘积是唯一的。
如果 $S$ 能够被拆分成两个质数之和（例如根据哥德巴赫猜想，任何大于 2 的偶数都可以），那么 S 就不敢说这句话，因为 P 拿到的乘积有可能正好是那两个质数的乘积。
空间坍缩：S 手中的 $S$ 必定不是偶数，而是一个奇数，且 $S-2$ 不能是质数（因为质数 2 和 质数 $S-2$ 的乘积必定只有唯一的分解方式）。通过这一步，我们可以穷举出 $S$ 只能是以下候选数字之一（如 11, 17, 23, 27, 29, 35... 等）。剔除所有 $x+y$ 不在这个候选 S 集合中的数对。

### 3. P 说“那我现在知道了”
逻辑推导：P 听到 S 的话后，也得出了上述的候选 S 集合。P 去检查自己手中那个确定的乘积 $P$ 的所有可能分解方式。他发现，在这些乘积分解方式对应的所有“和”里面，有且仅有一个和落在了这个候选 S 集合中。
空间坍缩：剔除所有在候选 S 集合中有多种可能性的乘积 $P$。此时，P 已经锁定了唯一的 $(x, y)$。

### 4. S 说“那我现在也知道了”
逻辑推导：S 听到 P 的话后，知道 P 已经通过步骤 3 找到了唯一解。S 去检查自己手中那个确定的和 $S$ 的所有可能的加和拆分方式。他发现，在这些拆分方式中，有且仅有一对 $(x, y)$ 能够让 P 在步骤 3 中得出唯一解 (在 S 和 P 共享了候选集合 $S$ 后)。
空间坍缩：在候选的和 $S$ 中，寻找具有唯一幸存加和拆分方式的 $S$。

## 结论
经过上述推演，整个样本空间只剩下一个唯一的数对：
x = 4, y = 13
此时 S = 17, P = 52。

Q.E.D.
