# 康威跳棋军队 (Conway's Soldiers)

本目录包含了康威跳棋军队的可达性分析与不变量证明。

---

## 一、系统的形式化定义 (Formal Definitions)

在一个无限大的二维整点网格 $\mathbb{Z}^2$ 上进行单人状态机博弈。

### 1. 状态空间与初始条件

- 设定不可跨越的初始分界线为 $y = 0.5$。
- **初始状态**：棋子（士兵）仅能放置于半平面 $y \le 0$（即 $y = 0, -1, -2 \dots$）内的网格点上。每个坐标点至多容纳 1 个棋子。初始配置的棋子总数必须为**有限值**。半平面 $y > 0$ 初始为空。

### 2. 移动规则（独立钻石棋/Peg Solitaire 规则）

- 严禁棋子发生任何形式的“滑动”或“平移”。
- 设存在三个相邻且共线的网格点 $A, B, C$（仅限水平或垂直方向）。当且仅当 $A$ 与 $B$ 被棋子占据，且 $C$ 为空时，允许发生一次**合法跳跃**：$A$ 处的棋子跃过 $B$ 落在 $C$。
- 操作完成后，$C$ 处状态更新为占据，同时 $A$ 与 $B$ 处的棋子被移除。每次移动必然导致系统总棋子数减一。

### 3. 目标状态

- 通过有限次合法移动，使得至少有一个棋子到达目标行 $y = 5$（即点 $(0, 5)$ 及其等价位置）。

---

## 二、物理限制与低阶状态的可达性 (Reachability for $y \le 4$)

在规则的强制约束下，棋子的任何推进必须以牺牲相邻棋子作为“跳板”为代价。这种物理限制导致了状态空间在向高阶目标推进时发生指数级膨胀。

### 命题 1：到达 $y=1$ 至少需要 2 个初始棋子。

- **构造**：于 $(0, 0)$ 和 $(0, -1)$ 放置棋子。
- **操作**：$(0, -1)$ 跃过 $(0, 0)$ 落于 $(0, 1)$。战损为 1，达成目标。

### 命题 2：到达 $y=2$ 至少需要 4 个初始棋子。

- **构造与推演**：登陆 $y=2$ 的必要前置条件是在同列的 $y=1$ 与 $y=0$ 均存在存活棋子。

  1. 先动用 2 个棋子（如上述命题 1）占领 $(0, 1)$，消耗后 $(0, 0)$ 为空。
  2. 因禁止平移，必须动用另外 2 个棋子（如置于 $(-2, 0)$ 与 $(-1, 0)$），通过横向跳跃落入 $(0, 0)$ 进行填补。
  3. 最终 $(0, 0)$ 跃过 $(0, 1)$ 登陆 $(0, 2)$。总消耗为 4 个棋子。

### 推论

由于不断需要底层横向跳跃以“运送和铺路”，克服空间拥挤导致的路径消耗极高。数学上可证，到达 $y=3$ 需 8 个棋子，到达 $y=4$ 需 20 个棋子。

## 三、单调量（势能函数）的代数构造动机 (Motivation for the Monovariant)

为了证明 $y=5$ 的不可达性，我们引入不变量方法。核心思想是寻找一个**单调递减量（Monovariant）**——即一个状态映射函数，使得系统的“总势能”在任何合法操作下均不增加。

### 权重的推导过程

假设对于距离目标点 $d$ 步的网格，其权重为 $x^d$，其中 $x \in (0, 1)$ 为未知的衰减因子。

根据移动规则，棋子向目标方向跳跃时（从距离 $k$ 跃过 $k-1$ 到达 $k-2$），系统损失了 $x^k$ 与 $x^{k-1}$ 的势能，新增了 $x^{k-2}$ 的势能。为保证总势能非增，必须满足不等式约束：

$$x^{k-2} \le x^{k-1} + x^k$$

由于 $x > 0$，等式两边同除以 $x^{k-2}$，得到系统物理规则在代数上的核心约束：

$$1 \le x + x^2$$

为了使证明边界达到极限紧凑，同时使向前的有效跳跃实现**势能守恒**，我们取该不等式的临界等式：

$$x^2 + x - 1 = 0$$

解此一元二次方程，取其正根，即得黄金分割率的倒数 $\varphi$：

$$\varphi = \frac{\sqrt{5} - 1}{2}$$

此代数根并非主观设定，而是由博弈的跳跃规则严格推导出的唯一度量因子。

## 四、康威定理及规范数学证明 (Theorem and Formal Proof)

## 定理 (Conway's Theorem)

对于任意有限数量的初始棋子配置，不存在任何有限步的合法移动序列可以使得棋子到达目标行 $y = 5$。

## 证明

### 1. 势能函数定义

设目标点为 $T = (0, 5)$。定义网格上任意点 $S = (x, y)$ 到 $T$ 的曼哈顿距离为：

$$d(x, y) = |x| + |5 - y|$$

赋予每个点 $S$ 权重 $w(S) = \varphi^{d(x, y)}$。

在任一时刻，系统总势能 $V$ 定义为所有被占据网格点的权重之和：

$$V = \sum_{S \in \text{Occupied}} \varphi^{d(S)}$$

已知 $\varphi$ 满足代数恒等式：$\varphi^2 + \varphi = 1 \implies 1 - \varphi = \varphi^2$。

### 2. 引理 1：合法移动不增加系统总势能 ($\Delta V \le 0$)

考察一次合法跳跃，棋子从 $A$ 跃过 $B$ 落在空位 $C$。势能变化量 $\Delta V = w(C) - w(A) - w(B)$。分三种情形讨论：

- **向目标点逼近**：$d(C) = k-2$, $d(B) = k-1$, $d(A) = k$。

  $$\Delta V = \varphi^{k-2} - \varphi^{k-1} - \varphi^k = \varphi^{k-2}(1 - \varphi - \varphi^2) = 0$$

- **背离目标点**：$d(C) = k+2$, $d(B) = k+1$, $d(A) = k$。

  $$\Delta V = \varphi^{k+2} - \varphi^{k+1} - \varphi^k < 0$$

- **等距横移**：

  $$\Delta V = \varphi^k - \varphi^{k-1} - \varphi^k = -\varphi^{k-1} < 0$$

综上，任意合法移动后，$V_{\text{final}} \le V_{\text{initial}}$。

### 3. 引理 2：下半平面满载势能的理论极限 ($V_{\max} \le 1$)

计算 $y \le 0$ 半平面内所有网格点权重之和的无穷级数极限。

首先对纵轴 $x=0$ 求和（距离 $d = 5 - y$）：

$$V_{x=0} = \sum_{y=0}^{-\infty} \varphi^{5-y} = \sum_{n=0}^{\infty} \varphi^{5+n} = \frac{\varphi^5}{1 - \varphi} = \frac{\varphi^5}{\varphi^2} = \varphi^3$$

对称求和包含纵轴及两侧所有列（列 $x$ 的距离增量为 $|x|$）：

$$V_{\max} = V_{x=0} + 2 \sum_{x=1}^{\infty} \varphi^x \cdot V_{x=0} = \varphi^3 + 2 \sum_{x=1}^{\infty} \varphi^{x+3}$$

$$V_{\max} = \varphi^3 + 2 \left( \frac{\varphi^4}{1 - \varphi} \right) = \varphi^3 + 2\varphi^2$$

代入 $\varphi^2 + \varphi = 1$ 进行降阶化简：

$$V_{\max} = \varphi(\varphi^2) + \varphi^2 + \varphi^2 = \varphi(1 - \varphi) + 2\varphi^2 = \varphi - \varphi^2 + 2\varphi^2 = \varphi + \varphi^2 = 1$$

由于物理规则限制初始配置必须为**有限数量**的棋子，因此初始系统的实际势能必定严格小于该满载极限：

$$V_{\text{initial}} < 1$$

### 4. 矛盾导出

若存在合法移动使棋子到达目标点 $T(0, 5)$，则终态势能和中必然包含目标点自身的权重：

$$w(0, 5) = \varphi^0 = 1$$

由此推得，成功到达目标所需的系统终态总势能满足 $V_{\text{final}} \ge 1$。

结合引理 1 与引理 2，得出系统不等式链：

$$1 \le V_{\text{final}} \le V_{\text{initial}} < 1$$

显然 $1 < 1$ 构成了绝对的数学矛盾。

故无论采取何种有限的初始棋子配置与跳跃策略，到达 $y = 5$ 均是不可能的。

$$\blacksquare$$
